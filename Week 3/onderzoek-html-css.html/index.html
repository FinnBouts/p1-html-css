Onderzoek HTML & CSS


Waarom noem je de eerste HTML-pagina meestal index.html?

Webservers (zoals Apache, Nginx, IIS) hebben standaarden ingesteld: wanneer je het domein bezoekt zonder dat je een bestandsnaam opgeeft, zoekt de server automatisch naar een standaardbestand in de hoofdmap. Vaak is dat index.html (of index.php, index.htm, etc.).


Met welke versie van HTML werken we op dit moment (2025)?

We gebruiken HTML Living Standard van WHATWG als de actuele versie.


Met welke versie van CSS werken we op dit moment (2025)?

CSS kent niet één versie zoals “CSS3” in de zin van één pakket maar is opgedeeld in modules. Elke module evolueert: sommige modules zijn “stabiel”, andere in ontwikkeling.


Waarom ziet een website er soms net iets anders uit in verschillende browsers, zoals Chrome, Firefox, Edge of Safari?

Rendering engine verschillen Browsers gebruiken verschillende engines: Blink (Chrome, Edge, Opera, etc.), WebKit (Safari), Gecko (Firefox). Elke engine interpreteert HTML/CSS (en JavaScript) licht anders.


Welke drie webbrowsers zijn op dit moment het populairst? Noem ze in volgorde van gebruik/populariteit.

Google Chrome — domineert met ~ 65‑70 % wereldwijd. Apple Safari — tweede, vooral sterk op iPhones, iPads en Mac. ~ 15‑20 %. Microsoft Edge — derde plek, meestal een stuk lager dan Safari, maar groeit/combineert mee door Windows‑integratie etc. ~ 5‑6 %.
